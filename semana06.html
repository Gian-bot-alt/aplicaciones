Ejercicio 01: Cálculo de sueldo neto para vendedores

Enunciado: Una empresa paga a sus vendedores un sueldo bruto que se calcula sumando un sueldo básico más una comisión. El sueldo básico es S/. 350.75. La comisión es igual al 5% del importe total vendido en el mes. Por ley, todo empleado está sujeto a un descuento igual al 15% del sueldo bruto. Diseñe un algoritmo que imprima la boleta de un vendedor indicando el sueldo básico, la comisión, el sueldo bruto, el descuento y el sueldo neto.

Requerimientos Funcionales:

  • Calcular la comisión (5% de las ventas).
  • Calcular el sueldo bruto sumando el sueldo básico y la comisión.
  • Aplicar un descuento del 15% sobre el sueldo bruto.
  • Calcular el sueldo neto restando el descuento al sueldo bruto.

Código en Dart:

import 'dart:io';

class Vendedor {
  double sueldoBasico = 350.75;
  double ventas = 0.0;

  void ingresarVentas() {
    print('Ingrese el importe total vendido en el mes:');
    String? input = stdin.readLineSync();
    ventas = double.parse(input!);
  }

  double calcularComision() {
    return ventas * 0.05;
  }

  double calcularSueldoBruto() {
    return sueldoBasico + calcularComision();
  }

  double calcularDescuento() {
    return calcularSueldoBruto() * 0.15;
  }

  double calcularSueldoNeto() {
    return calcularSueldoBruto() - calcularDescuento();
  }

  void imprimirBoleta() {
    print('--- Boleta de Pago ---');
    print('Sueldo Básico: S/ $sueldoBasico');
    print('Comisión (5% de ventas): S/ ${calcularComision()}');
    print('Sueldo Bruto: S/ ${calcularSueldoBruto()}');
    print('Descuento (15% del sueldo bruto): S/ ${calcularDescuento()}');
    print('Sueldo Neto: S/ ${calcularSueldoNeto()}');
  }
}

void main() {
  Vendedor vendedor = Vendedor();
  vendedor.ingresarVentas();
  vendedor.imprimirBoleta();
}

Ejercicio 02: Distribución de inversión en una feria

Enunciado: Dado el monto total de dinero a invertir, diseñe un programa que determine cuánto gastará la empresa en cada rubro (alquiler, publicidad, transporte, etc.).

Requerimientos Funcionales:

  • Solicitar el monto total de inversión.
  • Calcular los gastos para cada rubro en base a los porcentajes establecidos.
  • Imprimir los valores distribuidos en cada rubro.

Código en Dart:

import 'dart:io';

void calcularGastos(double inversion) {
  var rubros = {
    'Alquiler de espacio': 0.23,
    'Publicidad': 0.07,
    'Transporte': 0.26,
    'Servicios feriales': 0.12,
    'Decoración': 0.21,
    'Gastos varios': 0.11
  };

  rubros.forEach((rubro, porcentaje) {
    print('Gasto en $rubro: S/ ${(inversion * porcentaje).toStringAsFixed(2)}');
  });
}

void main() {
  print('Ingrese el monto total de inversión:');
  double inversion = double.parse(stdin.readLineSync()!);

  calcularGastos(inversion);
}

Ejercicio 03: Descuento en la compra de camisas

Enunciado: Una tienda ofrece un descuento de 7% + 7%. Diseñe un programa que determine el importe de la compra, los descuentos y el importe a pagar.

Requerimientos Funcionales:

  • Solicitar el precio del producto y la cantidad de unidades.
  • Calcular el primer y segundo descuento.
  • Determinar el importe a pagar aplicando ambos descuentos.

Código en Dart:

import 'dart:io';

void calcularDescuentos(double precio, int cantidad) {
  double importeCompra = precio * cantidad;
  double primerDescuento = importeCompra * 0.07;
  double segundoDescuento = (importeCompra - primerDescuento) * 0.07;
  double totalDescuento = primerDescuento + segundoDescuento;
  double importePagar = importeCompra - totalDescuento;

  print('Importe de la compra: S/ $importeCompra');
  print('Primer descuento (7%): S/ $primerDescuento');
  print('Segundo descuento (7% sobre el resto): S/ $segundoDescuento');
  print('Total descuento: S/ $totalDescuento');
  print('Importe a pagar: S/ $importePagar');
}

void main() {
  print('Ingrese el precio de la camisa:');
  double precio = double.parse(stdin.readLineSync()!);

  print('Ingrese la cantidad de camisas:');
  int cantidad = int.parse(stdin.readLineSync()!);

  calcularDescuentos(precio, cantidad);
}

Ejercicio 04: Cálculo de sueldo con bonificación por hijos

Enunciado: El sueldo bruto de los empleados de una empresa se calcula sumando el sueldo básico más la bonificación por hijos. El sueldo básico se calcula multiplicando las horas trabajadas por la tarifa horaria. Además, el empleado tiene un descuento en base al sueldo bruto. Diseñe un programa que determine el sueldo básico, el sueldo bruto, el descuento y el sueldo neto que le corresponde.

Requerimientos Funcionales:

  • Solicitar la categoría del empleado (A o B) y las horas trabajadas.
  • Calcular la bonificación por hijos.
  • Calcular el descuento en base al sueldo bruto.
  • Determinar el sueldo neto.

Código en Dart:

import 'dart:io';

double calcularTarifaHoraria(String categoria) {
  if (categoria == 'A') {
    return 45.0;
  } else if (categoria == 'B') {
    return 37.5;
  } else {
    throw Exception('Categoría no válida');
  }
}

double calcularBonificacion(int hijos) {
  if (hijos <= 3) {
    return hijos * 40.5;
  } else {
    return hijos * 35.0;
  }
}

double calcularDescuento(double sueldoBruto) {
  if (sueldoBruto >= 3500) {
    return sueldoBruto * 0.135;
  } else {
    return sueldoBruto * 0.10;
  }
}

void main() {
  print('Ingrese la categoría del empleado (A o B):');
  String categoria = stdin.readLineSync()!.toUpperCase();

  print('Ingrese las horas trabajadas:');
  int horasTrabajadas = int.parse(stdin.readLineSync()!);

  print('Ingrese la cantidad de hijos:');
  int hijos = int.parse(stdin.readLineSync()!);

  double tarifaHoraria = calcularTarifaHoraria(categoria);
  double sueldoBasico = horasTrabajadas * tarifaHoraria;
  double bonificacion = calcularBonificacion(hijos);
  double sueldoBruto = sueldoBasico + bonificacion;
  double descuento = calcularDescuento(sueldoBruto);
  double sueldoNeto = sueldoBruto - descuento;

  print('Sueldo Básico: S/ $sueldoBasico');
  print('Bonificación por hijos: S/ $bonificacion');
  print('Sueldo Bruto: S/ $sueldoBruto');
  print('Descuento: S/ $descuento');
  print('Sueldo Neto: S/ $sueldoNeto');
}

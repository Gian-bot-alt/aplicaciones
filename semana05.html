Ejercicio 01: Sumatoria de números primos en un rango

Enunciado: Escribe un programa que solicite dos números y calcule la sumatoria de los números primos que existen entre esos dos valores. Utiliza un bucle for o while para recorrer los números en el rango y verifica si son primos.

Requerimientos Funcionales:

  • El programa debe solicitar dos números enteros que representen el rango.
  • El programa debe calcular la suma de todos los números primos dentro del rango dado.
  • Debe manejar correctamente los límites del rango y verificar si los números son primos.

Código en Dart:

import 'dart:io';

bool esPrimo(int num) {
  if (num <= 1) return false;
  for (int i = 2; i <= num ~/ 2; i++) {
    if (num % i == 0) return false;
  }
  return true;
}

void main() {
  stdout.write('Ingrese el límite inferior: ');
  int inicio = int.parse(stdin.readLineSync()!);

  stdout.write('Ingrese el límite superior: ');
  int fin = int.parse(stdin.readLineSync()!);

  int sumaPrimos = 0;

  for (int i = inicio; i <= fin; i++) {
    if (esPrimo(i)) {
      sumaPrimos += i;
    }
  }

  print('La suma de los números primos en el rango es: \$sumaPrimos');

  for (int i = 0; i < n; i++) {

    siguiente = a + b;
    a = b;
    b = siguiente;
  }
}

  int n = int.parse(stdin.readLineSync()!);

  fibonacci(n);

Ejercicio 3: Factorial de números grandes

Enunciado: Escribe un programa que calcule el factorial de un número grande (por ejemplo, 100) utilizando estructuras repetitivas y el tipo de datos BigInteger para manejar grandes números.

Requerimientos Funcionales:

  • El programa debe solicitar un número grande (por ejemplo, 100) e imprimir su factorial.
  • Debe manejar números grandes usando BigInt.

Código en Dart:

    import 'dart:io';
    
    BigInt factorial(BigInt n) {
      if (n == BigInt.one) return BigInt.one;
      return n * factorial(n - BigInt.one);
    }
    
    void main() {
      stdout.write('Ingrese un número para calcular su factorial: ');
      BigInt numero = BigInt.parse(stdin.readLineSync()!);
    
      BigInt resultado = factorial(numero);
    
      print('El factorial de \$numero es: \$resultado');

Ejercicio 4: Inversión de un número

Enunciado: Crea un programa que invierta los dígitos de un número entero ingresado por el usuario, utilizando un bucle while para extraer y reordenar los dígitos.

Requerimientos Funcionales:

  • El programa debe solicitar un número entero positivo.
  • Debe invertir los dígitos del número e imprimir el número invertido.

Código en Dart:

    import 'dart:io';
    
    void main() {
      stdout.write('Ingrese un número: ');
      int numero = int.parse(stdin.readLineSync()!);
    
      int invertido = 0;
    
      while (numero > 0) {
        int digito = numero % 10;
        invertido = invertido * 10 + digito;
        numero ~/= 10;
      }
    
      print('El número invertido es: \$invertido');

Ejercicio 5: Suma de matrices NxN

Enunciado: Escribe un programa que solicite dos matrices de tamaño N x N (donde N es proporcionado por el usuario) y luego realice la suma de las dos matrices utilizando bucles anidados for.

Requerimientos Funcionales:

  • El programa debe solicitar al usuario el tamaño de las matrices N x N.
  • Debe permitir ingresar los valores para dos matrices.
  • Debe sumar las matrices y mostrar el resultado.

Código en Dart:

    import 'dart:io';
    
    void main() {
      stdout.write('Ingrese el tamaño de las matrices (N): ');
      int n = int.parse(stdin.readLineSync()!);
    
      List> matriz1 = List.generate(n, (_) => List.filled(n, 0));
      List> matriz2 = List.generate(n, (_) => List.filled(n, 0));
      List> suma = List.generate(n, (_) => List.filled(n, 0));
    
      print('Ingrese los valores de la primera matriz:');
      for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
          stdout.write('Elemento [$i][$j]: ');
          matriz1[i][j] = int.parse(stdin.readLineSync()!);
        }
      }
    
      print('Ingrese los valores de la segunda matriz:');
      for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
          stdout.write('Elemento [$i][$j]: ');
          matriz2[i][j] = int.parse(stdin.readLineSync()!);
        }
      }
    
      for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
          suma[i][j] = matriz1[i][j] + matriz2[i][j];
        }
      }
    
      print('La suma de las matrices es:');
      for (int i = 0; i < n; i++) {
        print(suma[i]);
      }

Ejercicio 6: Número perfecto

Enunciado: Implementa un programa que encuentre y muestre todos los números perfectos entre 1 y 10,000. Un número perfecto es aquel que es igual a la suma de sus divisores propios. Usa un bucle para encontrar los divisores de cada número.

Requerimientos Funcionales:

  • El programa debe mostrar todos los números perfectos entre 1 y 10,000.
  • Un número perfecto es aquel que es igual a la suma de sus divisores propios.

Código en Dart:

    import 'dart:io';
    
    bool esPerfecto(int numero) {
      int sumaDivisores = 0;
    
      for (int i = 1; i < numero; i++) {
        if (numero % i == 0) {
          sumaDivisores += i;
        }
      }
    
      return sumaDivisores == numero;
    }
    
    void main() {
      print('Números perfectos entre 1 y 10,000:');
      for (int i = 1; i <= 10000; i++) {
        if (esPerfecto(i)) {
          print(i);
        }
      }

Ejercicio 7: Matriz en espiral

Enunciado:Crea un programa que imprima una matriz cuadrada de tamaño n x n en forma de espiral. Utiliza bucles anidados para recorrer las posiciones de la matriz en el orden adecuado.

Requerimientos Funcionales:

  • El programa debe solicitar al usuario el tamaño de la matriz cuadrada.
  • Debe generar y mostrar la matriz en forma de espiral.

Código en Dart:

    import 'dart:io';
    
    void main() {
      stdout.write('Ingrese el tamaño de la matriz: ');
      int n = int.parse(stdin.readLineSync()!);
      
      List> matriz = List.generate(n, (_) => List.filled(n, 0));
      
      int valor = 1;
      int top = 0, bottom = n - 1, left = 0, right = n - 1;
    
      while (top <= bottom && left <= right) {
        for (int i = left; i <= right; i++) {
          matriz[top][i] = valor++;
        }
        top++;
        for (int i = top; i <= bottom; i++) {
          matriz[i][right] = valor++;
        }
        right--;
        if (top <= bottom) {
          for (int i = right; i >= left; i--) {
            matriz[bottom][i] = valor++;
          }
          bottom--;
        }
        if (left <= right) {
          for (int i = bottom; i >= top; i--) {
            matriz[i][left] = valor++;
          }
          left++;
        }
      }
    
      print('Matriz en forma de espiral:');
      for (var fila in matriz) {
        print(fila);
      }

Ejercicio 8: Verificación de un número Armstrong

Enunciado:Escribe un programa que verifique si un número de n dígitos ingresado por el usuario es un número de Armstrong (o narcisista). Utiliza un bucle for para separar y elevar cada dígito a la potencia correspondiente.

Requerimientos Funcionales:

  • El programa debe solicitar un número entero ingresado por el usuario.
  • Debe verificar si el número se lee igual en ambas direcciones.

Código en Dart:

    import 'dart:io';
    
    void main() {
      stdout.write('Ingrese un número: ');
      String numero = stdin.readLineSync()!;
    
      String invertido = numero.split('').reversed.join('');
      if (numero == invertido) {
        print('El número \$numero es un palíndromo.');
      } else {
        print('El número \$numero no es un palíndromo.');
      }

Ejercicio 9: Cálculo de potencias usando multiplicación repetida

Enunciado:Crea un programa que calcule la potencia de un número usando multiplicación repetida, es decir, sin utilizar la función Math.pow(). El programa debe solicitar una base y un exponente, y luego calcular la potencia utilizando un bucle while o for.

Requerimientos Funcionales:

  • El programa debe solicitar una base y un exponente.
  • Debe calcular la potencia usando multiplicación repetida y mostrar el resultado.

Código en Dart:

    import 'dart:io';
    
    void main() {
      stdout.write('Ingrese la base: ');
      int base = int.parse(stdin.readLineSync()!);
    
      stdout.write('Ingrese el exponente: ');
      int exponente = int.parse(stdin.readLineSync()!);
    
      int resultado = 1;
      for (int i = 0; i < exponente; i++) {
        resultado *= base;
      }
    
      print('\$base elevado a la \$exponente es: \$resultado');
